# @1r21/api-h5

## Usage Note

For full exposed APIs, see `src/api.ts`. You can also run `TARGET=api-h5 pnpm build` from repo root, which will generate an API report at `temp/api-h5.api.md`.

## API

- `getNews`
- `getNewsById`
- `translate`
